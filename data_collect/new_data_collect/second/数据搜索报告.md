## 指标2：无人机实名登记率
**替代指标**：已登记无人机数量 / 无人机保有量估算值

### 数据来源
- 民航局无人机实名登记系统（UOM平台）

### 搜索结果：⚠️ 部分数据可查，但无法获取省级年度数据

### 官方数据源
| 链接 | 状态 |
|------|------|
| https://uom.caac.gov.cn | ⚠️ 需要登录，无公开API |
| https://www.caac.gov.cn | ✓ 可访问，但无相关数据 |

### 关键发现
1. **全国数据**（2025年）：全行业实名登记无人机数量272.6万架，操控员执照超27万本（来源：新浪新闻，2025年10月）
2. **UOM平台**：民用无人驾驶航空器实名登记系统于2017年开始实施
3. **接口规范**：MH/T 3030-2023标准规定了数据交换接口，但不对外开放

### 说明
- 按省份、年度的无人机登记数量**未公开**
- 系统数据仅向注册用户开放，**无可公开访问的JSON接口**

- 按省份的飞行小时数和事故数据**基本未公开**

## 指标4：低空监管平台覆盖率
**替代指标**：是否建成省级低空监管平台（0-1虚拟变量）

### 数据来源
- 各省市交通运输厅公开信息
- 各省市公安局公开信息

### 搜索结果：❌ 找不到2019-2020年的系统化省级数据

### 关键发现
1. **安徽省**：2024年1月正式上线"安徽省低空无人机管理与服务平台"，系国内首家省级平台
2. **广东省**：2025年6月上线"广东省低空飞行综合管理平台"
3. **其他省份**：2024-2025年陆续开始建设，**2020年前几乎无省级平台**

### 官方数据源
| 链接 | 状态 |
|------|------|
| https://www.mot.gov.cn/ | ✓ 可访问 |
| 各省交通运输厅网站 | ✓ 可访问 |

### 说明
- 2020年及之前：**几乎没有**省级低空监管平台
- 省级平台建设高峰在2024-2025年
- 2019-2020年数据**无法获取**

- 飞行任务审批涉及军航、民航多头管理
- 审批时长数据**未公开**

### 详细搜索结论

#### ❌ 无法获取的数据（2013-2023年30省份）

1. **临时飞行空域数量**
   - 原因：属于空管系统内部管理数据，未对外公开
   - 状态：所有年份、所有省份均无公开数据

2. **低空监管平台覆盖率**
   - 原因：省级低空监管平台基本在2024-2025年才大规模建设
   - 状态：2019-2020年及之前几乎无省级平台，无法获取有效数据

3. **飞行任务审批平均时长**
   - 原因：各省市低空飞行服务中心大多在2020年后成立
   - 状态：审批时长数据未公开，无统一公开标准

#### ⚠️ 部分可获取的数据

4. **无人机实名登记率**
   - 可获取：全国层面数据（2025年约272.6万架）
   - 不可获取：按省份、年度的细分数据
   - 来源：UOM平台需要登录，无公开API

5. **通用航空安全事故起数**
   - 可获取：全国层面年度数据（2016年23起，2017年38起，2021年16起）
   - 不可获取：按省份细分数据
   - 来源：民航安全信息统计报告

6. **通用航空经营许可证数量**
   - 可获取：区域层面数据（华北153家，华东222家等，截至2025年8月）
   - 不可获取：2013-2023年按省份的年度历史数据
   - 来源：民航局通用航空管理系统

### 建议

1. **指标1、4、6**：建议放弃或使用替代指标（如政策文件数量、是否有相关规划等）
2. **指标2**：可尝试通过学术论文或行业报告获取历年数据
3. **指标3**：可使用全国层面的事故率数据，按飞行小时数估算
4. **指标5**：可尝试联系民航局获取历史数据，或使用区域层面数据
