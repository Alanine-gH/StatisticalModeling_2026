# 1.研究主题：

本研究聚焦低空经济对绿色交通的赋能效应，构建双系统综合评价体系，通过融合统计计量与深度学习方法量化影响机制、预测发展趋势并优化资源配置，为区域低空经济与绿色交通协同发展提供可落地的决策支持。

# 2.主要内容：

本研究基于低空经济与绿色交通双维度指标体系，首先采用改进的时间衰减CRITIC-熵权组合赋权法，测度2016-2023年全国30个省份双系统的综合发展水平，刻画其时空演变特征并对比东部、中部、西部、东北四大区域的发展差异，解决传统静态赋权法对新兴产业早期数据代表性不足的问题。

其次，构建空间杜宾-多路径并行中介效应融合模型，先通过全局与局部莫兰指数验证双系统的空间自相关性，再分解低空经济对绿色交通的直接影响与间接溢出效应，同时检验三条核心传导路径的独立贡献度与交互影响，并按四大区域开展异质性分析，弥补传统面板回归模型忽略空间关联性与多路径交互作用的缺陷。

最后，基于影响机制分析结果，采用改进的自适应粒子群优化算法求解多目标资源配置模型，设置基准投入、加大投入、重点投入三种发展情景，按四大区域汇总生成差异化的最优低空经济资源配置方案，解决传统政策制定依赖经验判断、缺乏量化支撑的问题，为地方政府推动低空经济与绿色交通协同发展提供科学的决策依据。

# 5.研究方法：

## 问题1：

如何科学测度2016-2023年全国30个省份低空经济与绿色交通的综合发展水平

### **研究方法：**

采用改进的时间衰减CRITIC-熵权组合赋权法。

第一步对所有原始指标进行极差标准化处理，消除量纲影响。

第二步分别计算CRITIC权重（基于指标间的冲突性与离散程度）和熵权（基于指标的信息熵）。

第三步引入指数时间衰减因子，给2021年之后的低空经济数据赋予更高权重。

第四步将两种权重与时间衰减因子结合得到综合权重，最终计算全国30省份每年的双系统各自的综合发展指数。

第五步，按东部、中部、西部、东北四大区域对省份进行分组，计算各区域每年的平均综合发展水平，对比分析区域间的发展差距、增长速度及演变趋势。同时梳理各区域内部省份的排名变化，识别各区域的领先省份与落后省份。

### **研究意义：**

客观准确地刻画各省份低空经济下绿色交通的发展现状及时空演变特征，为后续的影响分析与预测提供可靠的基础数据，解决传统静态赋权法无法适配新兴产业快速发展特征的问题。

### **可制作图表**

1.双系统指标权重对比柱状图（展示 CRITIC 权重、熵权、综合权重的差异）

2.2016 年、2020 年、2023 年全国低空经济下绿色交通发展水平时空热力图

3.四大区域（东部、中部、西部、东北）双系统发展水平年度变化折线图

4.2023 年各省份双系统发展水平排名条形图

### **模型检验及图表**

1.权重合理性检验：计算本方法与传统CRITIC-熵权法、单纯熵权法得到的综合指数的皮尔逊相关系数，制作相关系数矩阵热力图

2.稳健性检验：调整时间衰减因子（分别取 0.9、0.95、0.98），计算不同衰减因子下的综合指数，制作对比折线图，验证结果稳定性

 

## 问题2：

低空经济如何影响绿色交通发展，是否存在空间溢出效应及核心传导路径

### **研究方法**：

构建空间杜宾-多路径并行中介效应融合模型。

第一步计算全局与局部莫兰指数，验证双系统的空间自相关性。

第二步构建空间杜宾模型，分解出低空经济对绿色交通的直接影响（本省低空经济对本省绿色交通的影响）和间接溢出效应（本省低空经济对周边省份绿色交通的影响）。

第三步构建多路径并行中介效应模型，同时检验“技术创新传导”“产业结构升级”“交通结构优化”三条核心路径的独立贡献度与交互影响。

第四步进行分区域异质性分析，对比东部、中部、西部、东北四大区域影响机制的差异。

### **研究意义**：

清晰揭示低空经济赋能绿色交通的内在逻辑与空间传导规律，弥补传统面板回归模型忽略空间关联性的缺陷，同时一次性完成多路径检验，大幅提升分析效率，为制定差异化的区域政策提供依据。

### **可制作图表**

1.2023年低空经济与绿色交通局部莫兰散点图（分四个象限展示空间集聚特征）

2.2023年全国双系统空间集聚类型分布图（高-高、高-低、低-高、低-低四种类型）

3.空间杜宾模型直接效应、间接效应、总效应对比柱状图

4.多路径并行中介效应路径图（标注每条路径的系数和贡献度）

5.四大区域核心影响系数对比雷达图（展示不同区域三条中介路径的贡献度差异）

6.四大区域直接效应与间接效应对比分组柱状图

### **模型检验及对应图表**

1.空间自相关检验：计算2016-2023年全局莫兰指数，制作年度变化折线图，验证空间相关性的显著性

2.模型选择检验：进行LM检验、LR检验、豪斯曼检验，制作检验结果汇总表，验证选择空间杜宾模型的合理性

3.稳健性检验：替换空间权重矩阵（从地理距离矩阵替换为经济距离矩阵），重新估计模型，制作系数对比柱状图

 

## 问题3：

如何优化低空经济的资源配置，实现绿色交通综合效益最大化

### **研究方法：**

采用改进的自适应粒子群优化算法求解多目标优化模型。

第一步基于问题二的影响机制结果，构建以绿色交通综合效益最大化为目标函数，以低空经济产业投入、基础设施投入、创新投入为决策变量。

第二步设置各省份财政承受能力、资源禀赋等约束条件，并加入“基准投入”“加大投入”“重点投入”三种发展情景。

第三步对标准粒子群优化算法进行改进，引入自适应惯性权重与学习因子，提升算法的收敛速度与全局寻优能力。

第四步求解模型，按东部、中部、西部、东北四大区域汇总结果，生成四大区域、不同发展情景下的最优低空经济资源配置方案。

### **研究意义：**

解决传统政策制定依赖经验判断、缺乏量化支撑的问题，为地方政府优化财政资金投向、提高资源利用效率提供科学的决策工具。

### **可制作图表**

1.改进粒子群算法与标准粒子群算法、遗传算法的收敛曲线对比图

2.基准情景下四大区域最优投入结构饼图

3.三种发展情景下四大区域绿色交通综合效益预测值对比柱状图

4.四大区域不同投入要素的边际效益曲线

5.典型省份最优投入方案对比雷达图

6.全国低空经济资源配置优化建议空间分布图

### **模型检验及对应图表**

1.算法有效性检验：对比三种算法的收敛速度、最优解质量、运行时间，制作对比表格和收敛曲线

2.敏感性分析：分别调整目标函数中各投入要素的权重系数，制作绿色交通综合效益变化折线图，验证结果的稳定性

3.可行性检验：将优化方案与各省份历史投入数据对比，制作投入偏差条形图，验证方案的可落地性

在问题三的资源配置优化部分，选取一个典型省份（比如山东省），展示我们的优化方法在该省份的具体应用效果，比如优化后绿色交通综合效益可以提升多少，资金使用效率可以提高多少。这部分只需要半页左右的篇幅，但能极大地提升研究的实用性和说服力
