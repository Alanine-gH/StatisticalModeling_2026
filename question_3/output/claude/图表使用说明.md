## 📝 论文写作建议

### 1. 图表引用格式
```
如图1所示，GM(1,1)模型在MAE指标上表现最优（0.016），验证了...
根据图2可知，东部地区在基准情景下2024-2030年发展指数将从0.382提升至0.389...
```

### 2. 图表标题规范
```
图1 不同预测模型在测试集上的 MAE、RMSE 与 R² 对比
图2 2024-2030年四大区域三种情景下绿色交通综合发展指数预测趋势
图3 山东省三种情景下绿色交通发展水平对比（2024-2030年）
```

### 3. 结果讨论要点
- **精度对比**：强调所选模型的优越性（R²、MAE、RMSE）
- **趋势分析**：描述各区域上升/下降趋势，解释政策效果
- **区域差异**：对比东中西部，分析原因（基础设施、产业结构等）
- **政策建议**：基于图表结果，提出差异化投入方案

## 📌 使用建议

1. **Word插入**：直接拖入文档，设置为"嵌入型"或"四周型环绕"
2. **LaTeX插入**：
   ```latex
   \begin{figure}[htbp]
     \centering
     \includegraphics[width=0.8\textwidth]{图1_模型精度对比.png}
     \caption{不同预测模型拟合精度对比}
     \label{fig:model_comparison}
   \end{figure}
   ```
